# Azure Resource Group & Cost Management Lab

This repository contains my submission for the Azure Cloud Fundamentals lab.  
The project demonstrates my ability to create and manage Azure resources, verify subscription details, review cost management tools, and document the Shared Responsibility Model.

# Project Overview

The goal of this lab was to:

- Verify access to an active Azure subscription
- Create a Resource Group and deploy resources
- Review Azure Cost Management tools
- Summarize the region used and explain the Shared Responsibility Model
- Publish all evidence in a public GitHub repository
